# Mefstore
Symulator sklepu z możliwością zarządzania produkami, koszykiem oraz generowaniem paragonu.

## Uruchamianie
Aby skompilować program w katalogu głównym należy uruchomić polecenie:
```bash
make
```

Aby uruchomić program w katalogu build należy uruchomić polecenie:
```bash
./build/Mefstore
```
lub uruchomić Mefstore.exe w katalogu build
